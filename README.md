# SimpleAccordionTest

accordion on native js for loftschool 2017


###Install

```
$ git clone https://github.com/sychuginaanna/SimpleAccordeonTest
$ cd SimpleAccordeonTest/
$ npm i
$ gulp build / gulp dev
```

<b>Run Dev:</b>
````
gulp dev
````

<b>Run Production:</b>
````
gulp build
````
